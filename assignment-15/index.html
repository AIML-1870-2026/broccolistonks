Scale Reference

The Torino Scale (0–10) is the public-facing impact hazard index: 0 means no concern, 10 is a certain global catastrophe. All known objects currently rate 0.

The Palermo Scale is a logarithmic technical measure comparing impact probability to the background rate of random impacts of similar energy. Values below −2 are routine; values near 0 or above warrant scientific attention.
